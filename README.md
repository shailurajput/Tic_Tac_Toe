# 🎮 Tic Tac Toe Game (React + Vite + TypeScript)

A modern and interactive **Tic Tac Toe game** built using **React, Vite, and TypeScript**.
This project includes multiple game modes, smooth UI, and a clean user experience.

---

## 🚀 Features

* 🎯 **Two Game Modes**

  * Play vs AI
  * Play with Friend

* 🤖 **AI Opponent**

  * Smart moves (can block and try to win)
  * Shows "AI is thinking..." state

* 👥 **Friend Mode**

  * Two players play on the same device
  * Turn-by-turn gameplay

* 🔄 **New Game Reset**

  * Fully resets board and state
  * Works instantly without bugs

* 📊 **Score Tracking**

  * Tracks wins (X / O) and draws

* 🎨 **Modern UI**

  * Gradient background
  * Smooth animations
  * Responsive design

## 📦 Installation & Setup

Follow these steps to run the project locally:

```bash
# Clone the repository
git clone <your-repo-url>

# Go into the project folder
cd tic-tac-toe

# Install dependencies
npm install

# Start development server
npm run dev
```

---

## 🌐 Build for Production

```bash
npm run build
npm run preview
```

## 🚀 Deployment (Vercel)

* Push your project to GitHub
* Import it into Vercel
* Select:

  * Framework: **Vite**

## 🎮 How to Play

### ▶ Play vs AI

* You are **X**
* AI is **O**
* Try to beat the AI or block its moves

### 👥 Play with Friend

* Player 1 = X
* Player 2 = O
* Alternate turns
